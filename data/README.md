This folder will contain RNA-seq datasets downloaded from NCBI GEO/SRA.

Dataset Source
NCBI GEO: GSE140343

Description
RNA-seq dataset containing 51 lung adenocarcinoma tumor samples and 49 paired normal lung tissues.

Organism
Homo sapiens

Purpose
This dataset will be used to identify differentially expressed genes and discover potential driver genes using machine learning and network topology analysis.
